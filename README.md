## ＊はじめに
　『伺か』のゴーストの『百合』に関連する一次・二次創作作品集です。
　この企画の作品は全て『二次創作』であり、実在の人物・団体・事件等とは一切関係ありません。

[ダウンロード](https://github.com/Yuriuka/Yuriuka2/releases/download/%E9%85%8D%E5%B8%83/Yuriuka2.zip)
　
## ＊閲覧方法
　『yuriuka2.zip』を解凍後、『index.html』を開いて閲覧下さい。  
　また、フレーム・CSS・Java scriptを使用していますので、対応していないブラウザの場合、正常に閲覧出来ない可能性があります。  
　『in』フォルダ内に各作品を格納しておりますので、ブラウザで見るのが億劫に感じた場合はそちらでも閲覧可能です。
